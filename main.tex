\section{Introduction}
Lasso regression \citep{tibshirani_1996} is a popular modeling approach due to its ability to simultaneously perform estimation and variable selection through penalization. In the typical linear regression setting, lasso models an $n$-dimensional vector of continuous outcomes, $\y$, using a linear combination of covariates recorded in an $n$ by $p$ design matrix, $\X$, and a coefficient vector, $\bb$, which is subject to an $L_1$ penalty. The lasso estimate of $\bb$ is then found by minimizing the objective function:
\begin{equation*}
Q_{\text{lasso}}(\bb) = \tfrac{1}{2n}||\y - \X\bb||^2 + \lam||\bb||_1
\end{equation*}
A desirable property of the lasso is that its estimate of $\bb$ is \textit{sparse}, where some of its components are exactly zero for sufficiently large values of the penalty parameter, $\lambda$. For a given value of $\lambda$, variables whose corresponding coefficient is estimated as non-zero are deemed to be \textit{selected} by the lasso, a potential signal that they share a meaningful relationship with the outcome. 

However, in certain applications focusing on selection by the lasso is unsatisfactory. When nominal categorical variables are used as covariates they must be represented by dummy variables in the design matrix, but the lasso may select only some of them, with the selections differing depending upon the encoding scheme used, making it difficult to assess the selection status and importance of the categorical predictor. Similarly, in some applications covariates are allowed to have non-linear effects using basis expansion representations in the design matrix. However, the lasso may only estimate non-zero coefficients for some columns involved in the basis expansion, complicating assessments of selection status and importance.
The group lasso \citep{Yuan2006} addresses some of these shortcomings using an $L_2$ penalty on groups of coefficients, prompting selection to occur at the group level. This facilitates either all or none of the dummy variables representing a categorical predictor, or all or none of the coefficients involved in a basis expansion, to be selected as a group. More formally, the group lasso estimates $\bb$ by minimizing:
\begin{equation*}
Q(\bb) = \tfrac{1}{2n}||\y - \X\bb||^2 + \sum_{j = 1}^{J}\lambda_j ||B_j||
\end{equation*}
Here the penalty parameter, $\lambda_j$, is typically set to $\sqrt{K_j}\lambda$, where $K_j$ is the number of columns contained in group $j$. This allows a single penalty parameter, $\lambda$, to control the sparsity in the model while accounting for differences in group size. 
The group lasso’s penalization framework is easily extended to generalized linear models, with logistic regression \citep{meier2008} and other likelihood-based models, such as Cox regression \citep{Wang2009}, serving as prominent examples. It can also be generalized to other penalty functions, including non-convex penalties \citep{Breheny2012}, and penalties that enforce sparsity at both the group and individual coefficient levels \citep{Simon2013b}. \citet{Huang2012} provides a selective review of these methods.

A limitation of the group lasso and related methods is the lack of inferential tools to evaluate the statistical reliability of selected groups, and issue compounded by the fact that cross-validation, perhaps the most common method for tuning $\lambda$, is prone to favor models containing false positive selections due its focus on predictive accuracy \citep{Kim2017}. Selective inference, a framework for performing reliable statistical inference after variable selection has occurred, has been extended to the group lasso \citep{Yang2016}, but the method tends to be overly restrictive in the false discovery rate control it provides \citep{Miller2023} and its software implementation only selections forward stepwise selection. The \textit{knockoff filter} method of false discovery rate control \citep{barber2015, candes2018} has also been extended to the group-sparse models \citep{Dai2016}, but the approach also lacks a software implementation tailored to group lasso models. Computational approaches focused on group-level inference, such as the parametric bootstrap approach implemented in the {\tt EAinference} R Package \citep{Zhou2017}, focus on the uncertainty in the individual coefficient estimates of the group lasso, rather than the selections of entire groups, limiting their utility in regards to false discovery rate control.

With these limitations in mind, our paper focuses on methods to assess the reliability of group-level selections made by the group lasso model and its extensions. We propose methods for controlling the less restrictive marginal false discovery rate \citep{Breheny2019} of grouped selections made by these models, including the ordinary group lasso, group-penalized logistic regression and other generalized linear models, group-penalized Cox regression, as well as models using the group minimax concave penalty (MCP). Our proposed methods are accessible, computationally efficient alternatives compared to existing inferential tools used to assess group-level selections, and we demonstrate their robustness across a variety of data structures, focusing on scenarios involving nominal categorical predictors and non-linear basis expansions.
\section{Background}

\subsection{Group lasso regression}

Consider data of the usual form, $(\y, \X)$, where $\y$ records response values for $i \in \{1, \ldots, n\}$ independent observations, and $\X$ is an $n$ by $p$ design matrix. We focus on situations where the columns of $\X$ can be naturally organized into $J$ nonoverlapping groups, such that $\X = \{\X_1, \X_2, \ldots, \X_J\}$, where $\X_j$ denotes the $n$ by $K_j$ matrix containing the columns belonging to group $j$. 

We consider applications where the explanatory variables in $\X$ can be related with $\y$ using a probability model involving a set of coefficients, $\bb$, a well-known example being the standard linear regression framework, which specifies the relationship:
\begin{equation}
\label{m:lin}
\y = \X\bb + \beps
\end{equation}
where $\beps$ is a $n$-dimensional vector of errors that are independent and identically Normally distributed with mean 0 and variance $\sigma^2$. 
Under the model described in Equation~\ref{m:lin}, the group lasso solution, $\bbh$, is the minimizer, with respect to $\bb$, of the following objective function, denoted $Q(\bb)$:
\begin{equation}
Q(\bb) = \tfrac{1}{2n}||\y - \X\bb||^2 + \sum_{j = 1}^{J}\lambda_j ||\bb_j||
\end{equation}
where $\lambda_j$ is a penalty imposed on the $l_2$-norm of the coefficients corresponding to variables in group $j$. 

While it is possible to specify $\lambda_j$ separately for each group, the more common choice use $\lambda_j = \sqrt{K_j}\lambda$, which penalizes each group in proportional to its size using the shared parameter, $\lambda$. To aid in the presentation of our methods, we will assume $\lambda_j = \sqrt{K_j}\lambda$; however, this choice is not critical in any of our derivations or analyses.

The underlying framework of the group lasso can be generalized to a wide range of loss functions and penalization schemes. These variants can be expressed using the following objective function:
\begin{equation}
Q(\bb) = L(\bb|\X,\y) + \sum_{j = 1}^{J} p_{\lambda}(\bb_j)
\end{equation}
where $p_{\lambda}(\cdot)$ is a penalty function applied to each group of coefficients, and $L(\bb|\X,\y)$ is a loss function, which is based upon the log-likelihood in the case of generalized linear models, and the Cox partial likelihood in the case of Cox regression.
In their original proposal, \citet{Yuan2006} assume orthonormalization within each group, such that $\tfrac{1}{n}\X_j^T\X_j = \I$ for all $j \in \{1, \ldots, J\}$ with $\I$ representing the identity matrix, prior to model estimation. Although data are unlikely to naturally occur in this form, groups can be orthonormalized as a pre-processing step. Provided $K_j < n$ for all $j \in \{1, \ldots, J\}$, any optimization of the objective function performed using the orthonormalized data is equivalent to one done using the original data. The solution found on the orthonormal scale can also be easily converted back to the original scale. Throughout the remainder of this paper, we assume data have been orthonormalized within each group as a pre-processing step. See \citet{Simon2012} for further discussion of this topic.


\subsection{Marginal false discovery rates}

Following the seminal work of \citet{BH_1995}, false discovery rate (FDR) control has become a widely adopted inferential paradigm in applications involving large numbers of statistical comparisons. There is an enormous body of literature on FDR control, with many authors using slightly different definitions. We adopt the straightforward definition where the FDR is the expected number of false selections divided by the total number of selections, or the fraction of features/tests deemed ``significant” that are expected to be false positives. Under this definition, the FDR is controlled at $\alpha \%$ if no more than $\alpha \%$ of significant results are expected to be false positives.
Most research on FDR control focuses on \textit{large-scale univariate testing}, or applications involving the aggregation of results from a large numbers of univariate hypothesis tests that are essentially interchangeable. \citet{Farcomeni2008} and \citet{Strimmer2008} provide a more detailed overviews of these methods. However, our focus is on FDR control in the framework of regression modeling, where the notion of a false discovery is complicated by the presence of relationships between predictors. 

To better understand these complexities, consider the causal diagram shown below, which depicts a possible relationship between three explanatory variables, $A$, $B$, and $C$, and an outcome variable $Y$.

Variable $A$ has a direct causal relationship with $Y$ and should never be considered a false discovery, while variable $C$ is independent of $Y$ and should always be considered a false discovery. Whether or not variable $B$ should be considered a false discovery less clear. In large scale univariate testing approaches, variable $B$ is not be viewed as a false discovery because it is not \textit{marginally independent} of variable $Y$, which is the criteria evaluated in univariate hypothesis testing. However, in the regression setting, the coefficient in the data generating model corresponding to variable $B$ is zero, suggesting variable $B$ should be considered a false discovery.

The treatment of variables, or groups of variables, like $B$ in FDR control procedures can be approached in several ways. Under \textit{marginal FDR control}, any variable or group is considered a false discovery if it is deemed significant despite being fully independent of the outcome, or $\X_j \independent Y$. Under \textit{fully conditional FDR control}, a variable or group is a false discovery if it is deemed significant despite being independent of the outcome conditional upon all other variables in the data: $X_j \independent Y | X_{k \neq j}$. In penalized regression, it is common for only a subset of covariates to be actively contributing in a given model, which allows for \textit{pathwise conditional FDR control}. This perspective focuses on the model where $X_j$ first becomes active, and it conditions only on the other variables present in that model (a set we denote $M_j$) when assessing whether $j$ is a false discovery, or $X_j \independent Y | X_k \text{ for } k \in M_j$.

Our proposed methods use the less restrictive \textit{marginal false discovery rate} definition. While we acknowledge that the selection of variables like $B$ in the causal diagram can be undesirable in certain applications, it is often difficult, if not impossible, to determine the true causal structure of $A$, $B$, and $Y$ after the data have been collected. Additionally, it is always useful, regardless of the application, to limit the number of noise variables, like $C$, that are declared significant. For these reasons, along with the existing widespread adoption of marginal FDR control in realm of large-scale univariate testing, we believe that the marginal false discovery rate is a valuable quantity to consider in applications involving the group lasso and its variants. Furthermore, \citet{Breheny2019, Miller2019, Liang2021} have applied such a definition to the ordinary lasso, penalized generalized linear models and survival models, and penalized transformation models respectively.

\section{Marginal false discovery rates for the group lasso}
\label{Sec:main_meth}

\subsection{Linear regression}

Consider the data generation model:
\begin{equation}
\y = \X\bb + \beps
\end{equation}
where $\beps$ is a $n$-dimensional vector of errors that are independent and Normally distributed with mean 0 and variance $\sigma^2$. Furthermore, presume $\X$ has a known grouping structure that $\bb_j$ is a vector of length $K_j$ representing the model coefficients associated with group $j \in \{1, \ldots, J\}$. 

Our goal is to characterize the expected number and rate of false selections in a given group lasso model. We begin with the group lasso solution, $\bbh$, defined as the minimizer of the group lasso objective function, $Q(\bb)$:
\begin{equation}
Q(\bb) = \tfrac{1}{2n}||\y - \X\bb||^2 + \sum_{j = 1}^{J}\lambda_j ||B_j||
\end{equation}
Solving for $\bbh$ involves the subdifferential of $Q$ with respect to $\bb_j$:
\begin{equation}
\label{eq:kkt}
\begin{alignedat}{2}
-\tfrac{1}{n}\X_j^T(\y - \X_{-j}\bb_{-j}) + \bb_j + \lambda_j\tfrac{\bb_j}{||\bb_j||} &\quad \text{if } \bb_j &\neq 0 \\
-\tfrac{1}{n}\X_j^T(\y - \X_{-j}\bb_{-j}) + \lambda_j\mathbf{v} &\quad \text{if } \bb_j &= 0
\end{alignedat}
\end{equation}

Here, $\mathbf{v}$ is any vector satisfying $||\mathbf{v}|| < 1$, and $\X_{-j}$ denotes portion of $\X$ that remains after removal of the columns corresponding to the $j^{th}$ covariate group, with $\bb_{-j}$ similarly describing the associated model coefficients. In the convex optimization literature, the criteria shown in Expression~\ref{eq:kkt} are known as the KKT conditions. 

From the KKT conditions, it follows that if the $j^{th}$ group is selected into the model with non-zero coefficients, it must be that the case that:
\begin{equation}
\label{eq:post_kkt}
\tfrac{1}{n}\X_j^T(\y - \X_{-j}\bbh_{-j}) - \bbh_j = \lambda_j\tfrac{\bbh_j}{||\bbh_j||}
\end{equation}

\begin{theorem}
\label{Thm:main}
For the group lasso solution $\bbh$, the component $\bbh_j \ne \zero$ if and only if

\begin{equation}
\label{eq:selc}
\tfrac{1}{n}||\X_j^T(\y - \X_{-j}\bbh_{-j})||^2 > \lambda_j^2
\end{equation}

Further, assuming the group contained in $\X_j$ is marginally independent of $\y$, if $\tfrac{1}{n}\X_j^T \X_{-j}^T \inP \zero$ and $\lambda$ is chosen such that $\sqrt{n}(\bbh - \bb)$ is bounded in probability, then
\begin{equation}
\label{eq:thm1}
\tfrac{1}{n \sigma^2}||\X_j^T(\y - \X_{-j}\bbh_{-j})||^2 \inD \chi^2_{K_j}
\end{equation}
\end{theorem}

\begin{proof}
The first statement is a straightforward algebraic manipulation of Equation~\ref{eq:post_kkt} and a direct consequence of the KKT conditions. 

Then, expanding the left side of Expression~\ref{eq:selc} yields:
\begin{align}
\begin{alignedat}{2}
\tfrac{1}{n}||\X_j^T(\y - \X_{-j}\bbh_{-j})||^2 &= \tfrac{1}{n}||\X_j^T(\X\bb + \beps - \X_{-j}\bbh_{-j})||^2 \\
&= \tfrac{1}{n}||\X_j^T\beps - \X_j^T\X_{-j}(\bb_{-j} - \bbh_{-j})||^2 
\end{alignedat}
\end{align}
Noting $\tfrac{1}{\sqrt{n}} \X_j^T\X_{-j}(\bb_{-j} - \bbh_{-j}) \inP \mathbf{0}$ and $\beps \sim N(\zero,\sigma^2\I)$, we have $\tfrac{1}{n \sigma^2}||\X_j^T(\y - \X_{-j}\bbh_{-j})||^2 \inD \chi^2_{K_j}$.
\end{proof}

The first condition of Theorem~\ref{Thm:main}, $\tfrac{1}{n}\X_j^T \X_{-j}^T \inP \zero$, is satisfied when pairwise correlations between the columns of $\X_j$ and the other columns of $\X$ become negligible as $n$ increases. While the second condition, that $\sqrt{n}(\bbh - \bb)$ is bounded in probability, is met for suitable choices of $\lambda$, with additional details found in \citet{Liu2009}.

While it is unlikely in real data settings for all pairwise correlations between members of a group and the other covariates to be negligible, this assumption represents a valuable worst-case scenario for FDR control. Conceptually, when two groups involved in a penalized regression model are related, the selection of one group into the model reduces the chances that the other group is also selected. Thus, assuming groups are independent produces conservative estimates of the marginal false discover rate when relationships between variables exist. It is difficult to mathematically quantify the degree of conservatism introduced by such relationships, but we thoroughly explore this aspect of our methods via simulation studies in Section~\ref{Sec:sim}. Additionally, \citet{Breheny2019} provides a more detailed theoretical discussion of this topic in the context of the ordinary lasso.
\begin{clry}
\label{Clry:main}
Under the conditions outlined in Theorem~\ref{Thm:main}, for the group lasso model characterized by $\lambda$, the expected number of false discoveries and the expected rate of marginal false discoveries are bounded by:
\begin{align}
\label{eq:Mfdr}
\begin{alignedat}{2}
\FD &= \sum_{j = 1}^{J} \Pr \big(\chi^2_{K_j} > \frac{n\lam_j^2}{\sigma^2}\big)\\
\mFDR &= \frac{\FD}{\text{S}} 
\end{alignedat}
\end{align}
where $\text{S}$ denotes the total number of selected groups in the model characterized by $\lambda$.
\end{clry}

Theorem~\ref{Thm:main} implies the probability that the $j^{th}$ group is selected, given the group is marginally independent of the outcome, corresponds to the probability that a $\chi^2_{K_j}$ random variable is larger than $\frac{n\lam_j^2}{\sigma^2}$. In principle, we'd then sum over all groups that are marginally independent of the outcome to determine the expected number of falsely selected groups; however, the identity of these groups is unknown in practice, so summing overall $J$ groups produces a conservative alternative. But in many applications of the group lasso, the number of groups that are related to the outcome is presumed to be small in comparison to the total number of groups present in the data, making this effect relatively small.

Using these results, the process of calculating $\mFDR$ for the group lasso model defined by a given value of $\lambda$ is expressed in Algorithm~\ref{Alg:mfdr}.

\begin{algorithm*}
\caption{Calculating the mFDR upper bound}\label{Alg:mfdr}
\begin{algorithmic}[10]
\Procedure{}{}
\State Estimate $\sigma^2$ as $\hat{\sigma}^2$
\For {\texttt{$j \in \{1, \ldots, J\}$}}
\State $\widehat{\FD}_{j, \lam} = \Pr \big(\chi^2_{K_j} > \frac{n\lam_j^2}{\hat{\sigma}^2}\big)$ by the result of Theorem 1
\EndFor
\State $\widehat{\FD}_\lam = \sum_{j=1}^{J} \widehat{\FD}_{j, \lam} $
\State $\widehat{\mFDR}_\lam = \text{min}\left(\frac{\widehat{\FD}_\lam}{S_\lam}, 1 \right)$
\EndProcedure
\State \textbf{return} $\widehat{\mFDR}_\lam $
\end{algorithmic}
\end{algorithm*}

The initial step of Algorithm~\ref{Alg:mfdr} requires an estimate of $\sigma^2$, which can be found using the model’s residual sum of squares divided by its degrees of freedom or its cross-validated error.

\subsection{Generalized linear models}
\label{Sec:glm}

The group lasso penalty framework can be applied to other likelihood-based models, such as generalized linear models and Cox regression. Solving for $\bbh$ in these applications generally relies on a quadratic approximation of the log-likelihood (or Cox partial likelihood) function, $L$:
\begin{equation}
\label{eq:quad}
L(\be) = L(\bet) + (\be - \bet)^T\vv + \tfrac{1}{2}(\be - \bet)^T\W(\be - \bet)
\end{equation}
where $\eta = \X\bb$, with $\bet$ denoting the current estimate, and $\vv$ and $\W$ respectively denote the first and second derivatives of $L(\be)$ evaluated at $\bet$. It is worthwhile noting that $\W$ is a diagonal matrix in many commonly used models, including logistic regression.

Letting $\z = \bet - \W^{-1}\vv$, and dropping terms that are constant with respect to $\bb$, the approximation in Equation~\ref{eq:quad} leads to a loss function that is equivalent to weighted squared error loss:
\begin{equation}
L(\bb) \approx \tfrac{1}{2}(\z - \X\bb)^T\W(\z - \X\bb)
\end{equation}
Thus, the optimization approaches that solve for the group lasso solution, as well as their corresponding KKT conditions, can be adapted to likelihood-based models with the minor addition of the weight matrix, $\W$. 

Consequently, group selections made by these models are characterized by the following condition:
\begin{equation}
\label{eq:sel_glm}
\tfrac{1}{n}||\X_j^T\W(\z - \X_{-j}\bbh_{-j})||^2 > \lambda_j^2
\end{equation}
After applying the same series of steps previously described in the linear regression setting, we can approach the task of characterizing the marginal false discovery rate of these models using the left-hand side of Expression~\ref{sel_glm}:
\begin{equation}
\label{eq:Mfdr_proof_glm}
\tfrac{1}{n}||\X_j^T\W(\z - \X_{-j}\bbh_{-j})||^2 = \tfrac{1}{n}||\X_j^T\vv - \X_j^T\W\X_{-j}(\bb_{-j} - \bbh_{-j})||^2 
\end{equation}

\begin{prop}
\label{prop1}
Provided $(\X^T\W\X)^{-1/2}\X^T\vv \inD N(\zero, \I)$, where $\I$ is the $p$ x $p$ identity matrix, if $\tfrac{1}{n}\X_j^T \W \X_{-j}^T \inP \zero$, and $\sqrt{n}(\bbh- \bb)$ is bounded in probability, then the number and rate of marginal false discoveries of a likelihood-based model subjective to the group lasso penalty characterized by $\lambda$ can be estimated by:
\begin{align}
\label{eq:Mfdr_glm}
\begin{alignedat}{2}
\FD &= \sum_{j = 1}^{J} \Pr \bigg(\chi^2_{K_j} > \frac{n^2\lam_j^2}{\Tr(\X_j^T\W\X_j)}\bigg)\\
\mFDR &= \frac{\FD}{\text{S}} 
\end{alignedat}
\end{align}
\end{prop}

The first condition involved in Proposition~\ref{prop1} is a standard result of classical likelihood theory, which can be shown for many types of generalized linear models and Cox regression. The remaining conditions are direct analogs of those described in Theorem~\ref{Thm:main} for the linear regression setting. As a result, the derivations of the estimates in Equations~\ref{eq:Mfdr_glm} are analogous to those of Theorem~\ref{Thm:main} and Corollary~\ref{Clry:main}, with the primary distinction being the variance used in normalization. 
Like the linear regression setting, we can summarize the calculation of $\mFDR$ for given value of $\lambda$ in a group lasso penalized likelihood-based model using the following algorithm:

\begin{algorithm*}
\caption{Calculating the mFDR upper bound (GLMs}\label{Alg:mfdr_glm}
\begin{algorithmic}[10]
\Procedure{}{}
\State Estimate $\W \gets \nabla^2f(\beh)$
\For {\texttt{$j \in \{1, \ldots, J\}$}}
\State $\widehat{\FD}_{j, \lam} = \Pr \bigg(\chi^2_{K_j} > \frac{n^2\lam_j^2}{\Tr(\X_j^T\W\X_j)}\bigg)$ by Proposition 1
\EndFor
\State $\widehat{\FD}_\lam = \sum_{j=1}^{J} \widehat{\FD}_{j, \lam} $
\State $\widehat{\mFDR}_\lam = \text{min}\left(\frac{\widehat{\FD}_\lam}{S_\lam}, 1 \right)$
\EndProcedure
\State \textbf{return} $\widehat{\mFDR}_\lam $
\end{algorithmic}
\end{algorithm*}

Like the linear regression setting, these estimates will be conservative in the presence of pairwise correlations between columns of $\X$ corresponding to different groups, and in applications were $J$ is substantially exceeds the number of groups that are marginally independent of the outcome. 

Additionally, the estimates produced by Algorithm~\ref{Alg:mfdr_glm} are subject to further uncertainty arising from the use of the average diagonal element of $\X_j^T\W\X_j$ in normalization. Section~\ref{Sec:sim} explores the reliability of these estimates via simulation studies using group penalized logistic and Cox regression models under several different design matrices and correlation structures.

\subsection{Other penalty functions}

The general form of our derived $\mFDR$ estimators described is easily applicable to other penalization schemes related to the group lasso. A notable example is the group minimax concave penalty, or group MCP. In non-grouped applications, the MCP penalty function, $f_{\lambda, a}(\theta)$, is defined as:
\begin{equation}
\label{eq:mcp}
\begin{alignedat}{2}
\lambda\theta - \tfrac{\theta^2}{2a} &\quad \text{if } \theta &\leq a\lambda \\
\tfrac{1}{2}a\lambda^2 &\quad \text{if } \theta &> a\lambda
\end{alignedat}
\end{equation}
for values of $\lambda \geq 0$. Here, $a$ is a tuning parameter, with $a = 3$ being the default value used in most software implementations. MCP is a concave penalty where the degree of penalization diminishes, eventually becoming zero, for larger coefficients. 

In grouped applications, the composite group MCP estimate \citep{Breheny2012, Huang2012} is defined as the minimizer of:
\begin{equation}
Q(\bb) = \tfrac{1}{2n}||\y - \X\bb||^2 + \sum_{j = 1}^{J}f_{\lambda, b}\bigg(\sum_{k=1}^{K_j} f_{\lambda,a}(|\beta_{jk}|)\bigg)
\end{equation}
where the tuning parameter $b$ is typically chosen to be $K_ja\lambda/2$ to ensure that the group level penalty attains its maximum if and only if each of its components are at their maximum.

Although the coefficient estimates arising from group MCP regression tend to differ from those of the group lasso, the optimization conditions that define the selection criteria for a group becoming active in the model are identical. Consequently, the same estimators proposed in \ref{eq:Mfdr_glm1} and \ref{eq:Mfdr} can also be used to control the marginal false discovery rate for group MCP models. In Section~\ref{Sec:sim} we present results for both the group lasso and group MCP penalties in several of the simulation experiments we use to explore various aspects of our proposed $\mFDR$ estimators.

\section{Simulation experiments}
\label{Sec:sim}

\subsection{Data generation}

The following sections present results from a variety of simulation experiments investigating the validity and efficacy of our proposed $\mFDR$ methods. In all experiments, the covariates recorded in $\X$, are initially derived from numeric values randomly generated from a multivariate normal distribution with a mean of $\zero$ and a covariance matrix, $\Sigma_X$. The off-diagonal entries of $\Sigma_X$ are used to create dependencies between groups, and we focus on two particular cases: Independence - $cor(\x_a \x_b) = 0$, and Autoregressive - $cor(\x_a, \x_b) = \rho^{|a - b|}$, for all $a, b \in \{i,\ldots,J\}$. After these numeric values are generated, they are then used to construct design matrices for the two primary applications we focus on, models containing nominal categorical predictors and non-linear models using basis expansions.

In scenarios focused on categorical predictors, the final design matrix is constructed by binning the underlying numeric values into $k$ equally sized groups, which are then expressed in the design matrix using $k$ dummy variables. Outcomes are then generated from the linear model $\y = \X \bb+ \beps$, where $\epsilon_i$ is independently drawn from a $N(0,1)$ distribution in linear regression applications, from a Bernoulli distribution with $\Pr(y_i = 1) = 1/(1 + exp(-\x_i^T\bb))$ in logistic regression applications, or from an exponential distribution with a rate parameter equal to $exp(-\x_i^T\bb)$ in Cox regression applications.

In scenarios focused on non-parametric additive models, outcomes are generated directly from the underlying numeric values. For linear regression scenarios, this is done such that $\y = \sum_{j = 1}^{t} f(\x_j) + \beps$, where the function, $f$, defines a non-linear relationship between active features and the outcome, $t$ is a pre-specified number of non-noise features, and $\epsilon_i$ is independently drawn from a $N(0,1)$ distribution. For logistic regression scenarios, the outcome is drawn from a Bernoulli distribution with $\Pr(y_i = 1) = 1/(1 + exp(- \sum_{j = 1}^{t} f(\x_{ij})))$, and for Cox regression scenarios the outcome is drawn from an exponential distribution with a rate parameter equal to $exp(\sum_{j = 1}^{t} f(\x_{ij}))$. We consider two different forms of $f$: Quadratic - $f(X) = \beta X^2$, Piecewise Linear - $f(X) = \beta X$ if $X > 0$ and $f(X) = 0$ otherwise. Prior to model fitting, each feature undergoes a basis expansion with $K_j$ degrees of freedom, thereby yielding a design matrix containing $J$ groups of size $K_j$.

\subsection{False discovery rate control}

Standard software packages fit a series of group lasso models along a decreasing sequence of $\lambda$ values. Thus, the estimators from Equations~\ref{eq:Mfdr} and \ref{eq:Mfdr_glm} can be used to estimate the expected number of false discoveries contained in each model in this sequence. The curves displayed in Figure~\ref{Fig:lseq1} display the mean estimated number of marginal false group selections and the mean empirical number of false group selections averaged over 200 simulation repetitions along a fixed sequence of $\lambda$ values using the piecewise linear data generation scenario for group lasso penalized linear, logistic, and Cox regression with $n = 1000$, $J = 100$, and $t = 10$ under the independence correlation structure.

Figure~\ref{Fig:lseq1} demonstrates that the expected number of false selections as estimated via Equations~\ref{eq:Mfdr} and \ref{eq:Mfdr_glm} will, on average, accurately reflect the empirical number of false selections along the entire sequence of $\lambda$ values, with only a very slight degree of over-estimation induced by incidental correlations between the columns of $\X$. In the linear regression scenario, the estimated and empirical curves are nearly indistinguishable. While in the logistic regression and Cox regression scenarios the impact of the additional sources of variability discussed in Section~\ref{Sec:glm} can be observed, though the practical implication of this additional conservativism is unlikely to be consequential.

However, at the locations along the $\lambda$ sequence that are likely of the greatest interest, those where the estimated $\mFDR$ is between 10\% and 20\%, the discrepancy between the estimated and empirical number of false selections is relatively small in absolute terms. 

As a specific example, at $\lambda = 0.064$ in the linear regression setting, the average estimated marginal false discovery rate is 11.2\%, while the average empirical false discovery rate is only 8.1\%. Furthermore, even at the values of $\lambda$ which exhibit the greatest discrepancies, the estimated and empirical marginal false discovery rate tend to be close enough to each other to lead to similar conclusions. For example, at $\lambda = 0.039$ in the linear regression setting, the average estimated number of false discoveries is 19.27, suggesting the data analyst should conclude that there are likely to be many false discoveries among the groups selected by the model. The observed empirical average number of false discoveries at this value of $\lambda$ is 12.94, which is consistent with this conclusion.

\subsection{Comparison with other regression-based methods}
\label{Sub:comp}

Our third simulation experiment explores the ability our proposed $\mFDR$ methods to correctly identify true positives in comparison with existing group-level inferential methods capable of controlling the false discover rate. We generate data using $n = 200$, consisting of $J = 100$ groups with $t = 10$ of those groups sharing a real relationship with the outcome. We focus on the linear regression setting under the independence correlation structure, and we present the average number of true positive selections across 50 simulation repetitions for range of different signal strengths, as defined by $\beta$, the coefficient of each feature in the data generating model, while controlling the false discovery rate at 10\%. In our comparisons, consider the following methods:

\begin{itemize}[leftmargin=*, labelindent=0.5cm]
\item \textit{Forward selection with selective inference} where the selective inference approaches implemented in the {\tt groupfsInf} function contained in the {\tt selectiveInference} R package \citep{Selective_Inference} are used to obtain $p$-values for sequentially added groups. The forward stopping rules developed by \citet{GSell2016} are then applied to these results to control the false discovery rate. 
\item \textit{Large-scale testing} where single group hypothesis tests are performed separately, and the results are aggregated to control the false discovery rate using the methods of \citet{BH_1995} as implemented in the {\tt p.adjust} function contained in base R.
\item \textit{Data splitting} where half of the available data is used to select groups via the group lasso with $\lambda$ chosen to minimize the 5-fold cross-validation error, and the other half is used to fit a classical least squares regression model involving the selected groups and perform analysis of variance tests on each group. The results of these tests are then aggregated to control the false discover rate using the methods of \citet{BH_1995}.
\item \textit{Selective inference via the ordinary lasso} where the {\tt selectiveInference} function from the{\tt selectiveInference} R package is applied to an ordinary lasso regression model fit to the original data (before undergoing basis expansion) and the false discovery rate is controlled using forward stopping rules. This approach is only considered in the non-linear modeling scenarios previously described as ``piecewise linear" and ``quadratic".
\item \textit{Knockoff filter via the ordinary lasso} where the {\tt knockoff.filter} function contained in the {\tt knockoff} R package \citep{barber2015} is applied to an ordinary lasso regression model fit to the original data (before undergoing basis expansion). This approach is also only considered in the scenarios previously referred to as ``piecewise linear" and ``quadratic".
\end{itemize}

\begin{figure} [!htb]
\centering
\includegraphics[width=.9\textwidth]{Fig4.png}
\caption{\label{Fig:power} The average number of true positives in response to $\beta$ for various methods of false discovery rate control applied to different linear regression applications involving $n =200$, $J = 100$, $t = 10$ and the independence correlation structure.}
\end{figure}

Figure~\ref{Fig:power} shows the ability of our proposed $\mFDR$ control approach to on average identify more true positives than all of the aforementioned methods. We also find that the approach is slightly more powerful when applied to models using the group MCP penalty, despite prior simulations suggesting it yields more conservative false discovery rate control under this penalization scheme. Large-scale testing, which also adopts the marginal false discovery definition, exhibits comparable performance when the signal size is relatively small, but falls short of $\mFDR$ for larger values of $\beta$. 

Forward selection with selective inference, which adopts the stronger pathwise perspective on false discoveries, is the only other approach that averages at least one true positive for the values of $\beta$ we consider. All other methods struggle to identify any of the variables that share a relationship with the outcome. These results highlight the advantages of our $\mFDR$ approach and its marginal perspective on false discoveries when it comes to limiting the number of noise selections without being overly restrictive.

\subsection{Comparison with cross-validation}

Cross-validation is a resampling procedure that is commonly used to determine the optimal values of the penalty parameter(s) in penalized regression applications. Most penalized regression software will perform $k$-fold cross-validation, which splits the data into $k$ subsets such that model evaluation is done on data that are different from those used in model fitting. Despite its widespread use and its ability to provide an estimate of out-of-sample model performance, cross-validation will not necessarily limit the number or rate of false discoveries.

\section{Real data case study}

\subsection{Data}

Lung cancer is among the leading causes of death in the United States and the world, with a high mortality rate that is in part attributable to a lack of effective diagnostic tools while the disease is still in its early stages. \citet{Spira2007} studied the histologically normal bronchial epitheliums of smokers, collecting RNA expression data for $p = 22,215$ genetic features using Affymetrix HG-U133A microarrays. Among the $n = 192$ participants, 102 were cases who had already developed lung cancer and 90 were controls who had not developed lung cancer. The goal of the study was to determine whether gene expression data obtained at bronchoscopy from smokers with suspicion of lung cancer could be used to identify lung cancer biomarkers.

\subsection{Methods}

In our analysis, we consider several different ways of using the high-dimensional genetic data to predict the binary outcome of case-control status. Our primary approach applies a basis expansion with 4 degrees of freedom to each genetic feature, creating a new design matrix, $\tilde{\X}$, that contains 88,860 columns, corresponding to 22,215 groups of size $K_j = 4$. We then use the $\mFDR$ estimator proposed in \ref{eq:Mfdr_glm} to select a group lasso model that controls the expected marginal false discovery rate at 10\%. 

For comparison, we conduct a large-scale univariate testing approach where separate logistic regression models are fit corresponding to each genetic feature, and these models are then summarized by a likelihood-ratio test comparing their fit to that of an intercept-only model. The Benjamini-Hochberg procedure is applied to the resulting set of $p$-values to control the false discovery rate at 10\%. This approach is done separately for both the original design matrix, $\X$, and the expanded design matrix, $\tilde{\X}$. We also model the data without considering non-linear associations by fitting a lasso regression model using the binomial likelihood and applying the methods of \citet{Miller2019} to control the marginal false discovery rate. Finally, we also include results for each of these penalized regressions methods where the value of $\lambda$ defining the model is determined using 5-fold cross-validation.

Among penalized regression approaches, the lowest cross-validated misclassification error is observed when the ordinary lasso used on the unexpanded design matrix; however, the estimated $\mFDR$ of this model is 100\%. Since this $\mFDR$ estimate is inherently conservative, a value of 100\% doesn't necessarily indicate that all these selections are noise; however, it does suggest that we cannot be confident in the reliability of these selections. For the ordinary lasso, controlling the marginal false discovery rate at 10\% limits the number of genetic features selected to only 10, while at the same time substantially increasing the cross-validated misclassification error. In contrast, when the group lasso is applied to the expanded design matrix, 21 different genetic features can be selected while controlling the marginal false discovery rate at 10\%, with a misclassification error that is much closer to the minimum achieved by the ordinary lasso at the cross-validated value of $\lambda$. 

The large-scale testing results displayed in Table~\ref{Tab:case-studies} highlight important practical advantages of the $\mFDR$ approach. Although these methods are all based the same marginal perspective on false discoveries, regression-based methods tend to naturally limit the number of highly correlated features that are deemed significant. That is, in situations where many features are strongly related with both each other and the outcome, penalized regression tends to select only a single representative from the group, while large-scale testing tends to select all of them. Thus, in high-dimensional applications like this one, large-scale testing produces an overwhelming amount of ``leads" that researchers must then filter, group, and manually assess before any practical conclusions can be reached. In contrast, regression-based methods like $\mFDR$ tend to yield a more manageable set of features that likely contains less redundancy and is easier to translate into practical conclusions.

\section{Discussion}

The ability to estimate the marginal false discovery rate of a group lasso model, or a related variant, is a useful tool in assessing the reliability of selected groups. This is particularly true when considering that many popular methods for deciding upon a group lasso model, such as cross-validation or other goodness of fit criteria, may favor models containing undesirable numbers of false discoveries. While the $\mFDR$ approach is intended to control the marginal false discovery rate, which uses a more relaxed definition than some other methods with similar goals, it also tends to limit the number of indirect (non-causal) feature selections to much greater extent than other approaches that use this same definition, such as large-scale univariate testing. This makes $\mFDR$ an attractive and versatile tool in a variety of settings.

Additionally, there are relatively few methods that provide group-level false discovery rate control that currently have readily available software implementations. The $\mFDR$ procedures described in this paper, namely those outlined in Algorithm's 1 and 2, are implemented in the {\tt mfdr} function that is currently available in a forked version of the `R` package {\tt grpreg} \citep{Breheny2009} which can be installed directly from \url{https://github.com/remiller1450/grpreg} using the {\tt devtools} `R` package. Implementations are available for the linear, logistic, and Cox regression models using either the group lasso or group MCP penalty. 

Considering the computational burdens associated with applications like the Spira case study, the time it takes to include $\mFDR$ estimation as part of a broader analysis using group lasso regression is minimal. Despite the expanded design matrix in this application including 88,860 columns corresponding to 22,215 groups, it takes only a few minutes on a standard laptop to estimate the number and rate of marginal false discoveries for an entire sequence of 100 $\lambda$ values.
Furthermore, the application of $mFDR$ estimation methods to the group lasso helps facilitate coherent statistical inference in applications involving nominal categorical predictors, or in applications seeking to identify non-linear effects. As seen in our analysis of the Spira case study data and the results of simulation experiments, this can result in the identification of more variables of interest.
